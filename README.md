# 📊 Superstore Sales Dashboard (Excel)

## Project Overview
This project analyzes Superstore sales data using Microsoft Excel to provide business insights through an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- KPIs

## Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Loss Orders KPI
- Sales by Category
- Profit by Region
- Sales by Segment
- State-wise Sales Analysis
- Interactive Slicers
